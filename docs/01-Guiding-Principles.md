# Guiding Principles

1. Learner First — every decision is judged by learning impact.
2. Coach, Not Judge — feedback must guide and encourage.
3. Platform, Not Project — build a reusable engine, validate with English.
4. Content Is an Asset — content lives in database/curriculum, not hard-coded files.
5. Learning Engine ≠ Learning Policy — the engine can run stages; policy decides when.
6. The Engine Never Knows the Subject — English is the first domain, not the limit.
7. Security by Design — multi-tenant, roles, audit and privacy are core design constraints.
8. Every Sprint Produces a Running Deliverable.
